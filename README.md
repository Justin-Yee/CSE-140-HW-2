# CSE-140-HW-2
Lab assignment for CSE 140 Lab 2

Create a program that allows for the input of Mips R-format instructions to output in a certian format

sample:

add s1, t0, t1

Operation: add
Rs: t0 (R8) 
Rt: t1 (R9) 
Rd: s1 (R17) 
Shamt: 0 
Funct: 32 
Machine code: 00000001000010011000100000100000
